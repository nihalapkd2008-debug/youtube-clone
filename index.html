1) CSS Conflict Fixed Using Specificity

This text becomes crimson because the selector ".conflict-section .highlight" has higher specificity than normal paragraph selectors.

CSS conflicts happen when multiple styles try to affect the same element. The browser checks specificity rules to decide which style should win.

p { color: blue; }

.highlight { color: green; }

.conflict-section .highlight { color: crimson; }
